# Go Formosa
擁有豐富山林資源的臺灣，您可以體驗各種地形與自然景觀，在 Go Formosa 找到可靠的夥伴一同探索，現在就出發吧！

<a href="https://apps.apple.com/app/id1501021100"><img src="https://i.imgur.com/eRntvcA.png" height="40"></a>
## Features

#### 紀錄探索地圖
在地圖繪製探索軌跡，並且紀錄距離與進行時間。
1. 地圖可搜尋關鍵字地標
2. 紀錄行進軌跡

<img src="https://github.com/ChiouBo/Go-Formosa/blob/master/GoFormosaDemo/TrackMap.png" width="" height="355" /> <img src="https://github.com/ChiouBo/Go-Formosa/blob/master/GoFormosaDemo/Track.png" width="" height="355" />

#### 完整步道資訊
可以搜尋到全臺山林步道資訊，即時更新步道路況。
1. 關鍵字搜尋全臺步道資訊
2. 可直接以步道創建探索活動

<img src="https://github.com/ChiouBo/Go-Formosa/blob/master/GoFormosaDemo/TrailList.png" width="" height="355" /> <img src="https://github.com/ChiouBo/Go-Formosa/blob/master/GoFormosaDemo/TrailInformation.png" width="" height="355" />

#### 創建多人活動
讓其他人知道你也想去，開始規劃活動與夥伴一同探索。
1. 建立探索活動讓其他使用者參加
2. 人數可設定

<img src="https://github.com/ChiouBo/Go-Formosa/blob/master/GoFormosaDemo/CampaignList.png" width="" height="355" />

#### 活動成員互動
和參加活動的成員進行對話，一起相互認識再出發。
1. 活動獨立聊天室
2. 多人即時對話

<img src="https://github.com/ChiouBo/Go-Formosa/blob/master/GoFormosaDemo/Chatroom.png" width="" height="355" />

#### 個人成就紀錄
走過的路都不會白費，你也是未來的探索高手。
1. 地圖探索的歷史紀錄
2. 參與活動與地圖探索的個人成就紀錄
3. 使用者的探索等級

<img src="https://github.com/ChiouBo/Go-Formosa/blob/master/GoFormosaDemo/ProfileHistory.png" width="" height="355" /> <img src="https://github.com/ChiouBo/Go-Formosa/blob/master/GoFormosaDemo/ProfileAchievement.png" width="" height="355" /> <img src="https://github.com/ChiouBo/Go-Formosa/blob/master/GoFormosaDemo/ProfileLevel.png" width="" height="355" />
